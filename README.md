# AlfaMood

#### *AlfaMood mobile app working on React Native*
----------------------------------------------------------------
To install:
  In the working directory run this lines in the command prompt
  1) git clone https://github.com/ZhanZhaksylyk/AlfaMood/
  2) cd AlfaMood
  3) npm install
  4) npm start
